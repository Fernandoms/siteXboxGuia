Xbox One - Sobre

Nesta parte do site você vai encontrar informações mais detalhadas sobre o projeto que nós desenvolvemos.

Objetivos

Nós fizemos este site com o objetivo de compartilhar nossos conhecimentos, jogos favoritos, noticias interessantes, ou seja basicamento tudo que achamos interessante no Xbox.

Publico Alvo

Este site é destinado a todos os que como nós, também são amantes do Xbox e tem interesse em tudo relacionado as versões mais novas ou antigas

Tecnologias Utilizadas

Para fazer este site foram utilizadas as seguintes tecnologias:

  • HTML5
  • Bootstrap
  • JQuery
  • CSS3
Restrições e Requisitos Mínimos para o Usuário

Para melhor experiência com este site, recomendamos que utilize o navegador sempre atualizado e preferencialmente Mozilla Firefox e Google Chrome e um monitor com resolução maior que 800x600.

Dificuldades Encontradas

As maiores dificuldades que encontramos para construção deste site foram a tentativa de criar um layout simples e eficiente e achar conteudos interessantes para todos os adoradores do Xbox.
